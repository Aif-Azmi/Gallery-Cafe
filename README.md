Developed a dynamic web-based food ordering system for Gallery Café to streamline online orders and menu management. The application allows customers to browse menu items, place orders, and manage their accounts through a responsive interface.

The frontend was built using HTML, CSS, and Bootstrap to ensure a clean and user-friendly design. The backend was developed using PHP to handle business logic, authentication, and order processing, while MySQL was used for structured data storage and management.

🔹 Key Features:

Customer Registration & Login

Browse Menu & Food Categories

Add to Cart & Place Orders

Order History & Management

Admin Panel to Manage Menu & Orders

🔹 Technologies Used:

Frontend: HTML, CSS, Bootstrap

Backend: PHP

Database: MySQL

Tools: Git & GitHub

This project demonstrates my understanding of full-stack web development using PHP-based backend systems, relational database management with MySQL, and responsive UI design.
